# Trabalhador Consciente – Landing Page Oficial

Landing page desenvolvida para divulgação do livro **"Trabalhador Consciente"**, de **Dr. Ivan Luiz Rossi Anunciato**.

## 📖 Sobre o Projeto

Este projeto consiste em uma landing page responsiva criada para apresentar o livro **Trabalhador Consciente**, uma obra voltada à conscientização de trabalhadores, empregadores e profissionais da área jurídica sobre acidentes de trabalho, direitos, responsabilidades e formas de reparação.

A página tem como objetivo divulgar o livro, destacar seus principais conteúdos e facilitar a aquisição das versões impressa e digital, além de oferecer um e-book bônus sobre auxílio-acidente.

---

## ✨ Principais Funcionalidades

* Apresentação do livro
* Destaque dos principais temas abordados
* Seção sobre o público-alvo
* Divulgação do e-book bônus
* Perfil profissional do autor
* Informações sobre formatos disponíveis
* Integração com WhatsApp para compra e contato
* Layout responsivo para desktop e dispositivos móveis

---

## 📚 Conteúdo do Livro

O livro aborda temas como:

* O que caracteriza um acidente de trabalho
* Direitos do trabalhador acidentado
* Estabilidade, indenizações e benefícios previdenciários
* Responsabilidade do empregador
* Prevenção de riscos no ambiente de trabalho
* Procedimentos médicos, administrativos e judiciais
* Casos práticos baseados na experiência profissional do autor

## 🚀 Executando o Projeto

Instale as dependências:

```bash
npm install
```

Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

Acesse:

```text
http://localhost:5173
```

---

## 📦 Build para Produção

Gerar build:

```bash
npm run build
```

Visualizar build localmente:

```bash
npm run preview
```
